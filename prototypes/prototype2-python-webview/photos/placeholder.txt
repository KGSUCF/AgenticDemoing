GERTRUDE SHELL — PHOTO INSTRUCTIONS
=====================================

Place the Gertrude & Marty portrait photo in this folder (photos/)
with this filename:

    gertrude_and_marty.jpg

(The beautiful WWII-era portrait of Gertrude and Marty together)

The app will display it inside the photo frame on the main board.

Supported formats: JPEG (.jpg / .jpeg) or PNG (.png)
Recommended size: at least 260 × 320 pixels (portrait orientation)

The app checks for files in this order:
  1. gertrude_and_marty.jpg
  2. gertrude_and_marty.png
  3. gertrude.jpg
  4. gertrude.png

If no file is found, a warm placeholder is shown — the app still runs normally.
Restart the app after copying the file here.
