Methodology & deployment

This tool applies distant reading techniques to four works by Isaac Newton, drawn from Project Gutenberg. Distant reading — the computational analysis of large text corpora — lets us detect patterns invisible to close reading: which words dominate each text, which vocabulary is shared, and how each work's thematic register differs from the others.

Corpus

  • Observations upon the Prophecies of Daniel and the Apocalypse of St. John (1733) — Newton's theological and prophetic interpretation of Scripture
  • Opticks, or A Treatise of the Reflections, Refractions, Inflections and Colours of Light (1704/1730) — experimental natural philosophy
  • The Chronology of Ancient Kingdoms Amended (1728) — Newton's reconstruction of ancient history using astronomical evidence
  • Philosophiae Naturalis Principia Mathematica (1687) — Latin treatise on mathematical mechanics; the only non-English text

Method

  • Preprocessing: texts lowercased, tokenised on word boundaries, function words and short tokens removed via custom English and Latin stopword lists
  • Frequency counts: raw token counts of content words; top-N selected per document
  • Shared-term heatmap: terms appearing in ≥ 2 documents, ranked by corpus-wide total frequency
  • Thematic clusters: manually curated seed-word lists (Theology, History, Natural Philosophy, Astronomy/Maths) — each document scored by proportion of cluster seeds present in its top vocabulary
  • Concordance (KWIC): keyword-in-context passages extracted directly from the source texts with ±10 token windows

A note on the Latin text

The Principia is Latin; its term frequencies are not directly comparable to the English documents. The heatmap and shared-term analysis treat it separately. The concordance searches include it, with passages in the original Latin.

GitHub Pages deployment

  1. Create a new repository on GitHub (e.g. newton-corpus)
  2. Add this file as index.html in the repository root
  3. Go to Settings → Pages in your repository
  4. Under Source, choose Deploy from a branch → main → / (root)
  5. Click Save. Your site will be live at https://<username>.github.io/newton-corpus/ within a minute.
  6. No build step, no dependencies, no configuration files needed — this is a single self-contained HTML file.
